机考 SAT 数学共 **44 题 / 70 分钟**（两个模块各 22 题）。约 75% 为选择题，25% 为手动填答（grid-in）。考试提供**公式参考表**，并内置 Desmos 计算器——但真正省时的是先把方程列对。

本笔记按**四大官方领域**整理：每个领域先讲重点，再配带解析的练习题。

---

## 1. Algebra（代数）

**重点**

- 斜截式：$y = mx + b$，$m$ 为斜率，$b$ 为 $y$ 轴截距。
- 两点斜率：$m = \dfrac{y_2 - y_1}{x_2 - x_1}$。
- 平行线斜率相等；垂直线斜率乘积为 $-1$。
- 方程组 $\begin{cases}a_1x+b_1y=c_1\\ a_2x+b_2y=c_2\end{cases}$：两线平行（斜率相同、截距不同）时**无解**，两线重合时**无穷多解**。

**练习 1.1.** 若 $3x - 5 = 2x + 7$，求 $x$。

> **解.** $3x - 2x = 7 + 5 \Rightarrow x = 12.$

**练习 1.2.** 一条直线过 $(2, 1)$ 与 $(6, 9)$，求其方程。

> **解.** 斜率 $m = \dfrac{9-1}{6-2} = 2$。代入 $y - 1 = 2(x - 2)$ 得 $y = 2x - 3.$

**练习 1.3.** $k$ 取何值时下列方程组**无解**？
$$y = 4x + 1, \qquad y = kx - 3$$

> **解.** 无解即两线平行但不重合：斜率相等、截距不同，所以 $k = 4.$

## 2. Advanced Math（高阶数学）

**重点**

- 求根公式：对 $ax^2+bx+c=0$，$x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$。
- 判别式 $b^2 - 4ac$：正→两实根；零→一根；负→无实根。
- 顶点式 $y = a(x-h)^2 + k$ 的顶点为 $(h,k)$；顶点横坐标 $x = -\dfrac{b}{2a}$。
- 指数法则：$x^a x^b = x^{a+b}$，$\dfrac{x^a}{x^b} = x^{a-b}$，$(x^a)^b = x^{ab}$，$x^{-a} = \dfrac{1}{x^a}$，$x^{1/2} = \sqrt{x}$。

**练习 2.1.** 解 $x^2 - 5x + 6 = 0$。

> **解.** 因式分解：$(x-2)(x-3)=0 \Rightarrow x = 2$ 或 $x = 3.$

**练习 2.2.** 函数 $f(x) = x^2 - 6x + 11$ 在何处取最小值？

> **解.** $x = -\dfrac{b}{2a} = -\dfrac{-6}{2(1)} = 3.$（最小值 $f(3)=2$。）

**练习 2.3.** 若 $2^{x} = 32$，求 $x$。

> **解.** $32 = 2^5$，故 $x = 5.$

## 3. Problem-Solving and Data Analysis（解决问题与数据分析）

**重点**

- 百分比变化：$\dfrac{\text{新} - \text{旧}}{\text{旧}} \times 100\%$。
- 增加 $p\%$ 即乘 $\left(1 + \dfrac{p}{100}\right)$；减少则乘 $\left(1 - \dfrac{p}{100}\right)$。
- 平均数 $= \dfrac{\text{总和}}{\text{个数}}$；中位数为中间值；极差 = 最大 − 最小。数据越分散，标准差越大。
- 概率 $= \dfrac{\text{有利结果}}{\text{总结果}}$。

**练习 3.1.** 一件标价 \$80 的外套打 75 折（降价 25%），售价多少？

> **解.** $80 \times (1 - 0.25) = 80 \times 0.75 = \$60.$

**练习 3.2.** 一辆车 2.5 小时行驶 150 英里。按此速度，4 小时行驶多少英里？

> **解.** 速度 $= \dfrac{150}{2.5} = 60$ 英里/时。距离 $= 60 \times 4 = 240$ 英里。

**练习 3.3.** 数据 $4, 7, 7, 10, 12$，求平均数与中位数。

> **解.** 平均数 $= \dfrac{4+7+7+10+12}{5} = \dfrac{40}{5} = 8$；中位数为中间值 $= 7.$

## 4. Geometry and Trigonometry（几何与三角）

**重点（许多在参考表上）**

- 圆：面积 $= \pi r^2$，周长 $= 2\pi r$。
- 勾股定理：$a^2 + b^2 = c^2$。
- 特殊直角三角形：$45\text{-}45\text{-}90 \to$ 边长 $x, x, x\sqrt{2}$；$\;30\text{-}60\text{-}90 \to x, x\sqrt{3}, 2x$。
- 直角三角形三角比：$\sin\theta = \dfrac{\text{对边}}{\text{斜边}}$，$\cos\theta = \dfrac{\text{邻边}}{\text{斜边}}$，$\tan\theta = \dfrac{\text{对边}}{\text{邻边}}$。
- 一条直线上的角和为 $180^\circ$；三角形内角和为 $180^\circ$。

**练习 4.1.** 直角三角形两直角边为 6 和 8，求斜边。

> **解.** $c = \sqrt{6^2 + 8^2} = \sqrt{36+64} = \sqrt{100} = 10.$

**练习 4.2.** 一个圆面积为 $36\pi$，求其周长。

> **解.** $\pi r^2 = 36\pi \Rightarrow r = 6$。周长 $= 2\pi(6) = 12\pi.$

**练习 4.3.** 直角三角形中，角 $\theta$ 的对边为 3、斜边为 5，求 $\sin\theta$ 与 $\cos\theta$。

> **解.** $\sin\theta = \dfrac{3}{5}$。邻边 $= \sqrt{5^2 - 3^2} = 4$，故 $\cos\theta = \dfrac{4}{5}.$

## 使用方法

1. 先读一个领域的**重点**，再**不看答案**做练习。
2. 给每道错题标注原因（知识点不会 / 读错 / 时间失控 / 选项判断弱）。
3. 隔一天重做错题类型。某领域稳了，就到 Bluebook 限时做整段。
